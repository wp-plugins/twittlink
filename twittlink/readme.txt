== Description ==

Converts @choiz (or other twitter account) in Twitter's link. It's run in posts, pages and comments.

More info:

* [Twittlink] (http://www.choiz.fr/twittlink/).

== Installation ==

1. Download the plugin archive and expand it (you've likely already done this).
2. Copy the folder 'twittlink' into your wp-content/plugins/ directory.
3. Go to the Plugins page in your WordPress Administration area and click 'Activate' for twittlink.


* Download the plugin.
* Copy the folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
